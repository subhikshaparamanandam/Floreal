# Floréal

A premium single‑page website showcasing the **Floréal** AI‑powered plant and floral design platform.

## Features
- Grayscale glass‑morphism design
- Responsive layout with breakpoints for mobile
- Video background hero section
- Sections: Features, Gallery, Testimonials, CTA, Footer
- Simple scroll‑based navigation background change

## Getting Started
1. **Install a static server** (if you don't have one):
   ```bash
   npm i -g serve   # or npm i -g http-server
   ```
2. **Serve the project**:
   ```bash
   npm run dev
   ```
   This will start a local server (by default on http://localhost:5000) serving the `index.html`.
